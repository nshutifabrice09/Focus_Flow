# 📌 FocusFlow - Task Management System

**FocusFlow** is a robust task management system built with **Spring Boot** to help individuals and teams efficiently create, assign, and track tasks. It is designed for clarity, productivity, and collaboration.

## 🚀 Features

- ✅ Create, update, and delete tasks
- 👤 Assign tasks to users
- 📅 Track deadlines and status
- 🧑‍🤝‍🧑 User roles and authorization (Admin, Member)
- 💬 Comment on tasks for better collaboration
- 📁 Organize tasks by projects
- 🔒 Secure authentication with JWT (optional)
- 📊 Dashboard endpoints for analytics and summaries

## 🛠 Tech Stack

- **Backend:** Java, Spring Boot, Spring Data JPA, Spring Security (optional)
- **Database:** MySQL / PostgreSQL
- **Build Tool:** Maven / Gradle
- **API Docs:** Swagger / Springdoc OpenAPI
